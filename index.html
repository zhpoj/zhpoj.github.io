hexo的创建以及图片上传

1.图片显示插件命令下载命令

npm install https://github.com/CodeFalling/hexo-asset-image

2创建新博客文章命令

hexo new " "

3.typora进行编写文档
4.插入图片的时候需要，进行配置文件的编译_config.yml

post_asset_folder: ture

5.这样在进行新文件的创建的时候会出现保存图片文件的实现

6.打开.md进行博文的书写。在插入图片的时候路径是相对路径，同时将图片放到相对应的文件夹里。
7.总结常用的命令：g和d每次部署都要用到

hexo new "name"

hexo d 部署

hexo g 生成页面

hexo s 本地预览
